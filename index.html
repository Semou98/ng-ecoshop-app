Présentation du Projet

EcoShop - Plateforme E-commerce Écologique

EcoShop est une application web moderne conçue pour promouvoir des produits écologiques, offrant une expérience utilisateur fluide et intuitive.

Fonctionnalités Clés

  • Catalogue de produits écologiques
  • Système de panier dynamique
  • Navigation responsive
  • Intégration d'API pour les données produits

Technologies Utilisées

Angular TypeScript RxJS Bootstrap GitHub Vercel
